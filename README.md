# 🏦 Banking Loan Analytics Dashboard

End-to-End Data Analytics Project using MySQL & Power BI

---

## 📌 Business Problem

Financial institutions manage thousands of loan records across multiple branches, making it difficult to monitor portfolio performance, identify high-risk customers, track repayments, and analyze lending trends. This project converts raw banking data into an interactive analytics solution for data-driven decision-making.

---

## 📊 Project Highlights

📁 5 Relational Tables

🗃️ 5,000+ Loan Records Analyzed

👥 1,000+ Customer Records

🏦 Branch-wise Loan Analysis

📈 49 Business SQL Queries

📊 3 Interactive Power BI Dashboard Pages

📌 12+ Business KPIs

📉 Loan Portfolio Analysis

👥 Customer Segmentation Analysis

⚠️ Loan Risk Analysis

🏦 Branch Performance Analysis

📅 Loan Trend Analysis

🎛️ Page Navigation & Sync Slicers
